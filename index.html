Induktionsbeweis – Aufgabe 1.5 (a)

Zu zeigen ist:

∑k=1n+1 k·2k = n·2n+2 + 2    ∀ n ∈ ℕ0

1. Induktionsanfang:

Für n = 0:

Links: 1·2 = 2

Rechts: 0·2² + 2 = 2 ✅

2. Induktionsvoraussetzung:

Angenommen, die Aussage gilt für ein beliebiges n ∈ ℕ₀:

∑k=1n+1 k·2k = n·2n+2 + 2

3. Induktionsschritt:

Zu zeigen ist:

∑k=1n+2 k·2k = (n+1)·2n+3 + 2

Berechne linke Seite:

∑k=1n+2 k·2k = (∑k=1n+1 k·2k) + (n+2)·2n+2

Einsetzen der Induktionsvoraussetzung:

= n·2n+2 + 2 + (n+2)·2n+2 = (2n + 2)·2n+2 + 2

Faktorisiere:

= 2(n+1)·2n+2 + 2 = (n+1)·2n+3 + 2 ✅

✔️ Aussage bewiesen per vollständiger Induktion.
